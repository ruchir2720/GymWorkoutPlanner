# GymWorkoutPlanner

GymWorkoutPlanner is a modern Android fitness application designed to help users manage workouts, monitor fitness progress, calculate BMI, and maintain healthier daily routines through an intuitive and responsive mobile interface.

The project was developed as a practical Android development project focused on combining fitness management features with modern UI design, Firebase integration, and API-based functionality.

---

## Features

### User Authentication
- Firebase Authentication based login and signup system
- Secure user account handling
- User-specific data storage

### Workout Management
- Beginner-friendly workout sections
- Exercise planning and tracking
- Workout categorization for better organization

### BMI Analysis
- BMI calculator with health status indication
- Fitness tracking support
- Personalized user fitness insights

### Diet & Calorie Management
- Food search functionality using REST APIs
- Calorie-related information integration
- Diet planning support

### Progress Monitoring
- Fitness progress tracking interface
- Visual progress-based UI sections
- Goal-oriented fitness workflow

### Social / Friends System
- Firebase-based friend connection system
- User progress visibility between connected users
- Basic social interaction features

### Responsive UI Design
- Modern Android UI design using XML layouts
- RecyclerView-based dynamic components
- Multiple activity and navigation screens
- Ongoing optimization for cross-device responsiveness
